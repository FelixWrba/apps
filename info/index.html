Über AppHub

Die Internetseite AppHub stellt eine Sammlung an nützlichen und kostenlosen Web-Apps und Spielen bereit, welche zum Großteil quelloffen sind.

Datenverarbeitung

  • AppHub speichert Daten ausnahmslos lokal auf dem Endgerät des Nutzers
  • Es werden keine persönliche Daten an Dritte weitergegeben
  • Der gesamte Client-seitige Code ist einsehbar und wird in Deutschland von Felix Wrba entwickelt
